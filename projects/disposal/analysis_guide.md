
# 處置股產業分析指南 (Disposal Stock Industry Analysis Guide)

針對您的問題：「**處置股有好幾支，要怎麼看 Mean (平均)？**」

這是一個非常關鍵的問題。因為每支股票被處置的日期不同，股價水位也不同，我們不能直接平均股價。

必須透過以下步驟將它們「對齊」並「標準化」，才能算出有意義的平均值。

## 1. 核心概念：AAR (Average Abnormal Return)

我們看的「Mean」，通常是指 **「平均超額報酬 (Average Abnormal Return, AAR)」**。

### 步驟 A：計算單一個股的表現 (Abnormal Return)
對於每一支處置股，我們不看它「漲跌多少」，而是看它「贏過產業多少」。

$$ \text{個股超額報酬 (AR)} = \text{個股當日漲跌幅} - \text{產業平均漲跌幅} $$

- **例子**：
  - 當天您的處置股跌了 -3%。
  - 但同產業平均跌了 -5% (大盤不好)。
  - 您的 **AR** 就是 **+2%** (抗跌，表現優於產業)。

---

### 步驟 B：對齊時間 (Alignment)
既然要把不同日期的股票放在一起看平均，我們必須用「相對天數」：

- **s+0**：處置第一天
- **s+1**：處置第二天
- ...

---

### 步驟 C：計算橫斷面平均 (Cross-Sectional Mean)
這就是您問的 **「怎麼看 Mean」**。

把所有同產業的處置股，在 **「同一個相對時間點」** 的 AR 加總除以數量。

$$ \text{Mean AR (時間 t)} = \frac{\text{A股票在 t 時的 AR} + \text{B股票在 t 時的 AR} + ...}{\text{總股票數量}} $$

### 實作邏輯 (Python / Pandas)
如果您已經可以用程式取得資料，邏輯如下（這不是要您執行的代碼，是說明邏輯）：

1. **分組 (GroupBy)**：把資料依照 `[產業類別, 相對天數]` 分成小堆。
2. **聚合 (Aggregate)**：對每一堆算出 `mean`。

這樣您就會得到一張表：

| 產業 | 相對天數 (Day) | 平均超額報酬 (Mean AR) |
| :--- | :--- | :--- |
| 電子 | s+0 | -1.2% |
| 電子 | s+1 | -0.5% |
| 金融 | s+0 | +0.1% |

由這張表，您就可以畫出 **「各產業面對處置時的平均走勢圖」**。

## 2. 進階：累積平均 (CAAR)
有時候單看一天的平均會跳動，我們會把它累加起來 (Cumulative)。
- **CAAR** 曲線若是往下掉：代表該產業的處置股，隨時間經過，累積虧損越來越大（表現持續輸給產業）。

## 3. 進階驗證：產業指數作為連續指標 (Continuous Indicator)

如果您不想粗略地只分兩組，而是想直接把 **「產業指數」當作一個指標** 來觀察它與處置股的連動關係，建議使用 **「散佈圖 (Scatter Plot)」** 與 **「迴歸分析 (Regression)」**。

### A. 散佈圖分析 (Visualizing the Link)

這是最直觀能看到「產業」與「處置股」關係的方法：

- **X 軸 (指標)**：產業指數的累積漲跌幅 (例如 s+0 ~ s+5)。
- **Y 軸 (目標)**：處置股的累積漲跌幅 (同期)。
- **每一個點**：代表一次處置事件。

**如何解讀圖形？**
1.  **正相關 (斜率向上)**：點點分布成從左下到右上的帶狀。代表「產業漲越多，處置股就漲越多」。這直接驗證了您的假設。
2.  **檢視離群值 (Outliers)**：
    -   如果在 X 軸很右邊 (產業大漲)，但 Y 軸卻在下面 (處置股跌)，這些就是 **「處置真正有效」** 的案例。
    -   如果在 X 軸很右邊，Y 軸也很高，代表 **「處置無效，被產業帶飛」**。

### B. Beta 係數分析 (Quantifying the Effect)

您可以進一步計算 **Beta ($\beta$)** 值，量化產業對處置股的「牽引力」：

$$ R_{處置股} = \alpha + \beta \times R_{產業指數} + \epsilon $$

- **如果 $\beta \approx 1$**：處置股跟著產業連動，處置幾乎沒有「冷卻」這層關係。
- **如果 $\beta < 1$** (例如 0.5)：產業漲 10%，處置股只漲 5%。代表處置發揮了「煞車」作用。
- **如果 $\beta > 1$**：代表處置股是投機指標，比產業還瘋狂。

透過觀察 $\beta$ 在 **「處置前」vs「處置後」** 的變化，您就能精確回答：
> 「處置是否有讓這支股票脫離產業的炒作？」(預期處置後 $\beta$ 應該要變小)。

## 4. 每日動態追蹤 (Daily Analysis)

您問到：**「雖然看了產業影響，但我還是想看每一天的平均表現，這做得到嗎？」**

當然可以！我們可以把上述的分析 **「切成每一天」** 來看。這樣您就能同時看到「時間序列」與「產業影響」的動態變化。

### 方法一：每日相關係數走勢 (Correlation / Beta Evolution)

您可以算出每一天的產業連動程度：

- $T=0 (s+0)$：所有處置股當天漲幅 vs 產業當天漲幅 -> 算出 **Corr_0**
- $T=1 (s+1)$：所有處置股當天漲幅 vs 產業當天漲幅 -> 算出 **Corr_1**
- ...

畫出一條線：**X軸是時間 (s-5 ~ s+10)，Y軸是相關係數 (Correlation)**。

- **觀察點**：
  - 如果這條線在處置後 (s+0) **沒有掉下來**，依然維持高檔 (例如 0.8)，代表處置完全無效，股價每天都還是跟著產業在跳動。

### 方法二：多圖並列 (Facet Scatter Plots)

如果您想看細節，可以畫出 **「一連串的散佈圖」** (例如 6 張小圖)：
- **圖1 (s-1)**：處置前一天，點點排成一條線 (高度連動)。
- **圖2 (s+0)**：處置當天，點點開始變亂 (開始脫鉤)。
- **圖3 (s+1)**：處置第二天，點點變成一團迷霧 (完全脫鉤)。

這樣您不僅看到了「平均」，還看到了結構性的「關係變化」。

這種畫法最能一眼看出「處置的效果」，因為您可以直接比較 **兩條線的開口差 (Gap)**。

![個股 vs 產業 對照示意圖](/C:/Users/ownme/.gemini/antigravity/brain/8c745941-6133-4a99-a80f-5503490cd4f0/stock_vs_industry_comparison_1768653301732.png)

### 圖表解讀技巧
-   **藍色虛線 (產業基準)**：代表該段時間市場的「公定行情」。
-   **紅色實線 (處置股平均)**：代表處置股的實際表現。
-   **兩線差距 (Gap)**：即為 **Abnormal Return (超額報酬)**。

## 7. 關鍵思考：平均產業報酬有意義嗎？ (Interpretation)

您問到一個非常專業的問題：**「把不同時間、不同產業的報酬平均起來 (藍線)，這樣真的有意義嗎？」**

答案是：**非常有意義**，因為它能告訴您 **「處置事件的發生背景」**。

### 情境 A：藍線顯著大於 0 (例如平均 +0.5%)
- **含義**：這代表處置事件通常發生在 **「產業正熱的時候」** (Selection Bias)。
- **結論**：處置股不是單獨作亂，而是跟著產業一起雞犬升天。這時候處置通常效果較差 (因為風還在吹)。

### 情境 B：藍線接近 0
- **含義**：代表處置事件的分佈跟產業景氣無關，純粹是個股自己在亂。
- **結論**：這時候紅線的下跌，就是純粹的個股修正。

### 情境 B：藍線接近 0
- **含義**：代表處置事件的分佈跟產業景氣無關，純粹是個股自己在亂。
- **結論**：這時候紅線的下跌，就是純粹的個股修正。

所以，觀察 **「藍線的高度」** 本身就是一個重要的研究發現：它揭示了證交所抓處置股時，到底是抓到了 **「產業過熱 (Systemic Overheating)」** 還是 **「個股炒作 (Idiosyncratic Hype)」**。

## 8. 策略驗證：選「強勢產業」的處置股會比較好嗎？

回到您最核心的交易問題：**「我是不是應該專挑產業漲的處置股來買？」**

這可以透過 **「五分位分析 (Quintile Analysis)」** 來與歷史數據直球對決。

### 驗證步驟
1.  **排序 (Ranking)**：將歷史上所有的處置事件，依照 **「處置當下的產業漲跌幅」** 由高到低排序。
2.  **分組 (Bucketing)**：
    -   **Q1 組 (Top 20%)**：產業最強勢的一群 (例如產業漲 > 5%)。
    -   **Q5 組 (Bottom 20%)**：產業最弱勢的一群 (例如產業跌 < -5%)。
3.  **績效對決 (Performance Compare)**：
    計算這兩組在 **處置後 5 天 (s+5)** 的平均獲利。

### 判斷標準
-   如果 **Q1 組的平均獲利 >> Q5 組** (例如 +3% vs -4%)：
    -   **結論**：**Yes!** 產業強弱是處置股的保護傘。您應該只做多 Q1 組，避開 Q5 組。
-   如果 **Q1 組與 Q5 組獲利差不多** (例如都跌 -2%)：
    -   **結論**：**No.** 處置的效果太強，覆巢之下無完卵，產業再強也救不了。

### 實作建議
您可以直接用 Pandas 的 `pd.qcut` 指令來分組：
```python
df['ind_rank'] = pd.qcut(df['industry_ret_s0'], 5, labels=['Q5(Weak)', 'Q4', 'Q3', 'Q2', 'Q1(Strong)'])
df.groupby('ind_rank')['stock_ret_s5'].mean().plot(kind='bar')
```
這張長條圖若呈現 **「階梯狀上升」**，就證明了您的選股策略是正確的。

## 9. 結構釐清：每個分組都要看 s-5 ~ e+5 嗎？ (Trajectory Comparison)

您問到：**「所以每個分組內也都要有 s-5 ~ e+5 嗎？」**

答案是：**沒錯！這才是最完整的比較。**

我們不只是比「最後一天的獲利」，而是比 **「整條路徑」**。

### 想像一下您會畫出 5 條線 (Q1 ~ Q5)
- **X 軸**：時間 (s-5, s-4, ..., s+0, ..., s+10)
- **Y 軸**：平均累積報酬率

**您會看到的景象：**
- **Q1 線 (強勢產業)**：可能在處置後 (s+0) 稍微回檔一點點，然後就繼續往上衝。
- **Q5 線 (弱勢產業)**：在處置後 (s+0) 直接跳水，一路向南不回頭。
- **中間的 Q3 線**：就在這兩者之間。

### 這樣做的好處
這能讓您看出來 **「差異是在哪一天拉開的？」**
- 是在處置第一天 (s+0) 就決定命運？
- 還是前幾天大家差不多，直到第 5 天 (s+5) 強勢股才展現韌性？

- 還是前幾天大家差不多，直到第 5 天 (s+5) 強勢股才展現韌性？

所以，**「分組後的 Time-Series Plot (時間序列圖)」** 是驗證策略最精細的工具。

![五分位分析示意圖 (扇形展開)](/C:/Users/ownme/.gemini/antigravity/brain/8c745941-6133-4a99-a80f-5503490cd4f0/quintile_trajectory_schematic_1768653870346.png)

### 圖表解讀：
- 注意看 **s+0 (中間點)** 之後，這五條線是如何 **「扇形分開 (Fanning Out)」** 的。
- **最上面的紅線 (Q1)**：代表即使被處置，因為產業夠強，它照樣創新高。
- **最下面的藍線 (Q5)**：代表因為產業轉弱，處置變成壓垮駱駝的最後一根稻草。
- **中間的灰線 (Q3)**：代表無功無過，處置效果中性。

這張圖就是您向老闆或投資人證明 **「我選強勢產業是對的」** 最有力的證據。

## 11. 終極解答：熱力圖 (Heatmap)

您提到：**「長條圖只能看一天，好麻煩，能不能一張圖看全部？」**

這就是 Quant 最喜歡用的武器：**「熱力圖 (Heatmap)」**。它能把「時間 (Time)」和「分組 (Quintile)」全部濃縮在一張圖裡。

![五分位績效熱力圖 (Heatmap)](/C:/Users/ownme/.gemini/antigravity/brain/8c745941-6133-4a99-a80f-5503490cd4f0/quintile_heatmap_schematic_1768654076707.png)

### 如何秒懂這張圖？
-   **直觀感受**：看顏色就好！ **紅色代表賺錢，藍色代表賠錢**。
-   **橫向看 (時間)**：
    -   最上面那排 (Q1 強勢組)：隨著時間往右走，顏色 **越來越紅** (越賺越多)。
    -   最下面那排 (Q5 弱勢組)：隨著時間往右走，顏色 **越來越藍** (越賠越慘)。
-   **縱向看 (策略)**：
    -   在任何一天 (例如 s+5)，顏色都是 **上紅下藍**。這代表不管持有幾天，Q1 永遠比 Q5 好。

-   **縱向看 (策略)**：
    -   在任何一天 (例如 s+5)，顏色都是 **上紅下藍**。這代表不管持有幾天，Q1 永遠比 Q5 好。

一張圖，同時滿足了您對 **「時間路徑」** 與 **「策略勝率」** 的雙重檢驗。

## 14. 觀察結構質變：相關性衰退圖 (Correlation Decay Chart)

您提到：**「我不是要看賺不賺錢，我是想看這中間到底發生了什麼變化 (Change)？」**

這時候我們就不畫報酬率了，我們畫 **「相關係數 (Correlation Coefficient)」**。

這張長條圖顯示的是：**「處置股跟產業的關係有多親密？」**

## 16. 關鍵定義：怎麼判斷「強勢」還是「弱勢」？ (Criteria)

您問到了執行的核心：**「標準是什麼？怎樣算強？」**

針對處置股的特性（通常是短線過熱），我們推薦以下三種判斷標準：

### 方法 A：短期動能 (Short-term Momentum) [最推薦]
- **定義**：計算 **處置前 5 天 (s-5 ~ s-0)** 的產業指數漲跌幅。
- **判斷**：
    -   **強勢**：累積漲幅 > **+2%** (或 > 0%)。
    -   **弱勢**：累積漲幅 < **0%**。
-   **理由**：處置通常是因為「漲太多」，如果連產業都在漲，那這就是「真強勢」。

### 方法 B：相對強弱 (Relative Strength, RS)
- **定義**：看產業有沒有贏過大盤 (加權指數)。
    -   `RS = 產業漲幅 - 大盤漲幅`
-   **判斷**：
    -   **強勢**：RS > 0 (比大盤強)。
    -   **弱勢**：RS < 0 (比大盤弱)。
-   **理由**：這是最客觀標準，排除了大盤全面崩盤的雜訊。

### 方法 C：均線多排 (MA Trend)
- **定義**：看產業指數目前的均線排列。
-   **判斷**：
    -   **強勢**：產業指數 > **20日均線 (月線)**。
    -   **弱勢**：產業指數 < **20日均線**。
-   **理由**：適合波段操作者，確認這是一個上升趨勢中的回檔，還是下跌趨勢中的反彈。

**建議從「方法 A」開始嘗試，因為它與處置股的「過熱屬性」最直接相關。**

## 20. 參數微調：要用「漲幅%」還是「超額報酬 (Alpha)」？

您問得非常細膩：**「要不要減去大盤？」**

這取決於您認為 **「處置的成因」** 是什麼：

### A. 絕對漲幅 (Absolute Return) -> 推薦！
- **定義**：直接用產業漲幾% (例如 +10%)。
- **邏輯**：處置股的規定（警示、處置）通常是看 **「股價漲幅」** 或 **「週轉率」**，這些都是絕對數值。
- **優勢**：當全市場都瘋狂時（大盤漲10%，產業漲12%），這時候資金效應最強。如果您減去大盤只剩 +2%，反而會低估了當時的瘋狂程度。

### B. 超額報酬 (Excess Return / Alpha)
- **定義**：產業漲幅 - 大盤漲幅。
- **邏輯**：您想排除大盤波動，只看產業強弱。
- **劣勢**：處置機制不管大盤漲跌，只要個股噴太兇就會抓。

**結論**：針對處置股研究，我建議先用 **A (絕對漲幅)**，因為這跟證交所的遊戲規則比較一致。
